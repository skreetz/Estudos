# LAN (Local Area Network)

## Definição
Uma **LAN (Local Area Network)** é uma rede de computadores que conecta dispositivos dentro de uma área geográfica limitada, como uma **residência**, **escritório**, **escola** ou **edifício**.
## Características
- A**lta velocidade de comunicação**.
- **Baixa latência**.
- Administração local.
- Geralmente utiliza [[Ethernet]] ou [[Wi-Fi]].
## Componentes Comuns
- Hosts (computadores, smartphones, impressoras).
- [[Switch]].
- [[Roteador]].
- Interfaces de rede ([[NICs]])
## Endereçamento
- [[Endereço IP]] para comunicação na camada de rede.
- [[Endereço MAC]] para identificação das interfaces na camada de enlace.
## Exemplos
- Rede doméstica ou **LAN** conectando computadores e celulares.
- Rede corporativa dentro de um escritório.
- Laboratório de informática de uma escola.
## Vantagens
- Compartilhamento de recursos.
- Comunicação rápida entre dispositivos.
- Controle centralizado.
## Limitações
- Alcance geográfico restrito.
- Dependência da infraestrutura local.
