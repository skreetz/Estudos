# NIC (Network Interface Card)

## Definição
Uma **NIC (Network Interface Card)** é o componente de **hardware** ou **software** responsável por conectar um dispositivo a uma rede de computadores.
## Funções
- Enviar e receber quadros (frames) na rede.
- Implementar protocolos da camada de enlace.
- Fornecer um identificador único para a interface ([[NICs]]).
- Realizar a interface entre o dispositivo e o meio de transmissão.
## Tipos
### NIC com fio
- Utiliza cabos [[Ethernet]].
- Conector RJ-45.
- Comum em desktops, servidores e switches.
### NIC sem fio
- Utiliza [[Wi-Fi]] ([[IEEE 802.11]]).
- Comunicação por radiofrequência.
- Comum em notebooks, smartphones e IoT.
### NIC virtual
- Criada por software.
- Utilizada por **máquinas virtuais**, **contêineres** e **VPNs**.

Cada **NIC** possui um identificador chamado [[Endereço MAC]]
## Observações
- Um dispositivo pode possuir várias NICs.
- **Cada NIC possui seu próprio [[Endereço MAC]]**.
- O endereço MAC **identifica** a interface de rede, não necessariamente o **dispositivo** inteiro.
## Exemplos
- Placa Ethernet integrada à placa-mãe.
- Adaptador USB Wi-Fi.
- Interface de rede de uma máquina virtual.
